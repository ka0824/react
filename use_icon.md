> ## icon 사용하기


프로젝트 파일에 라이브러리 설치하기

```
npm install @mui/material @emotion/react @emotion/styled

npm install @mui/icons-material;

```

---------------------------------

사용할 라이브러리 불러오기

```
import SearchIcon from '@mui/icons-material/Search';    // 예시

<SearchIcon>   // 태그 형태로 사용

```
